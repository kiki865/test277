# AGENTS.md

## Project
MOSA怪獸挑戰師大附中：國中會考英文單字與句子挑戰介面。

## Rules
- Keep `data/words.js` as the source of 300 words.
- Keep `data/sentences.js` as the source of 60 sentences.
- Do not inline image assets into `index.html`. Use `assets/`.
- Preserve answer-gating logic: user must answer before moving to next word/sentence.
- Sentence word chips are hidden before answering; option pronunciation is available before answering; Chinese translation and grammar note show after answering.
